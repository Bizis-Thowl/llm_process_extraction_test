# Testing LLMs on their capability to extract process information

- To work, this Repository requires a arizephoenix instance. 
- A description for running arizephoenix under Docker can be found under [this link](https://arize.com/docs/phoenix/self-hosting/deployment-options/docker).
- Before starting: 
    - Make sure to install the [required packages](/requirements.txt).
    - Create a .env-file with the data that is listed in the .env-example
    - Create a folder with the local data you want to work with.
- Run [llm-process-extraction.py](/llm-process-extraction.py) to start
